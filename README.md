# trip_django
## 여행 sns

### 개발기간: 2023.02.01 ~ 2023.02.08


### 사용기술: python, django, sqlite3, docker, postgreSQL, Nginx


### 개발목적: 여행을 많이 다녀봤는데, 사진으로만 기록하기 아쉬워 글과 같이 정리하고자 여행sns를 개발하게 되었습니다.
